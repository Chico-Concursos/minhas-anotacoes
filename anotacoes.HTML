Gerencie suas anotações integradas com Google Sheets

Como usar

  1. Crie uma cópia da planilha modelo no Google Sheets
  2. Desenvolva um Web App no Google Apps Script
  3. Configure as permissões de acesso
  4. Cole o ID da planilha e o nome da página
  5. Implante como aplicativo web
  6. Insira a URL do Web App no parâmetro ?webAppUrl=SUA_URL

Este aplicativo permite criar, editar e excluir anotações, que são armazenadas diretamente em uma planilha do Google Sheets.
